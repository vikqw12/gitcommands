базовые git команды:
 
git init - инициал. локального репозитор.
git add - либо имя файла , либо поставить просто точку - добавление файла в индексацию.
git rm < имя файла > -- cached - удаление файлп из индексации.
git commit - m "сообщение" 
git log - посмотреть commit, который есть
git checkout <имя ветки> - переключиться на другую ветку
git dranch -d <имя ветки> уделение ветки
git merge <имя ветки> это вливание в текущую ветку указанной ветки
